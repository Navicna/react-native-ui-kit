# React Native Ui Kit

Ui kit for react-native developers

## Tools

- React-Native
- React
- Styled Components

## Usage

```js
import { StyledView, StyledText } from 'react-native-dev-ui';

function App() {
  return (
    <StyledView bgColor="white" alignItems="center" justifyContent="center" flex={1}>
      <StyledText>Hello World!!!<StyledText>
    </StyledView>
  );
}
```
